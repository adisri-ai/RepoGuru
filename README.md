# Project Overview  
RepoGuru is a RAG-based AI Agent that can be used by users to retrieve valuables insights/responses for any public GitHub repositories upon sharing the repo link. 
# Project Tech Stack  
1. **AI-Integration** : Langchain
2. **Web Framework**  : Streamlit
3. **Source Code**    : Python
4. **LLM Model**      : Ollama(Free-Tier)
